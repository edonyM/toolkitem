# PDF FILE CONVERTOR
------------

Convert files into pdf files from `*.txt`, `*.rst`, `*.jpg`, `*.png`.

* **txt2pdf.py** convertor from ['.txt', '.rst',...(plain text files)] to pdf.

* **img2pdf.py** convertor from ['.png', '.jpg'] to pdf.

* **pdfconvertor.py** merge all pdfs from convertors below.
